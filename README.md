# E-commerce API - Minimal implementation

## What is included
- Express + Mongoose project implementing models and routes for: users, products, categories, carts, orders, reviews.
- JWT auth (login/register), role checks (admin).
- Aggregations and example protected routes per the assignment.
- Error handling middleware.

## Quick start
1. Install Node.js (>=16) and MongoDB.
2. Copy `.env.example` to `.env` and edit values.
3. `npm install`
4. `node server.js`
5. Use Postman/Insomnia to test routes at `http://localhost:3000/api/...`

## Notes
- This is a minimal project meant for study and testing. Improve validations and add tests for production.
